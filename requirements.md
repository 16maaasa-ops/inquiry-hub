# マルチチャネル通知 + AI 分類 提案書

## 1. 現状の課題
- 月 750 件の問い合わせがメール / LINE / 電話に分散
- 営業時間外の LINE 取りこぼし
- カテゴリ別の対応速度がバラつき、クレーム対応が遅延

## 2. 提案する解決策
**MVP（5 営業日）**:
- Gmail / LINE → Slack に自動集約
- AI が 4 カテゴリに分類して、Slack チャネルに振り分け
- クレーム検出時は 5 分以内に営業部長個人 LINE へ通知

**Phase 2 ロードマップ**:
- 電話の音声テキスト化（Twilio 連携）
- SLA レポート自動生成

## 3. 期待される効果
- 取りこぼし率: 推定 15% → **0%**
- クレーム認知遅延: 平均 4 時間 → **5 分以内**
- 営業時間外も即対応可能

## 4. 費用・スケジュール
- 初期費用: **14.8 万円（税込）**
- 月額: **約 1.5 万円**（Vercel Pro + Supabase Free + LINE/Slack/Claude API 使用量）
- 納期: 5 営業日
